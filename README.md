# Lampa
Forked from [yumata/lampa-source](https://github.com/yumata/lampa-source)

## Install & Run

```bash
git clone https://github.com/androschuk/lampa-source.git
cd lampa-source
npm install
npm run start
```

## Documentation

Internal documentation can be generated locally:
```bash
npm run doc
```
